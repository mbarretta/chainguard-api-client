# chainguard-api-client

Fast, lightweight, type-safe TypeScript client for the [Chainguard API v2](https://edu.chainguard.dev/chainguard/api/spec-api-v2/).

Generated types from the upstream OpenAPI spec, a thin auth-aware wrapper over `openapi-fetch`, zero runtime overhead beyond native `fetch`.

> Status: targets API v2beta1. The full README with usage examples lands in T5.
